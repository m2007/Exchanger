# Exchanger
Показывает информацию о криптовалютах в реальном времени. Считает обмен между криптовалютами и валютами.

## Установка и запуск
Открыть терминал
```
git clone https://github.com/m2007/Exchanger.git
cd ./Exchanger
npm install
```
Открыть на http://localhost:8080/
```
npm run serve
```
Скомпилировать
```
npm run build
```
